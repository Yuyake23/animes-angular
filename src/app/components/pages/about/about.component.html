Sobre

Este site é um trabalho desenvolvido como objetivo parcial de nota para a disciplina de Programação WEB I, ministrada pelo professor Jorcivan Ramos. A disciplina faz parte do 5° perído do curso de Bacharelado em Sistemas de Informação.

Os animes estão mockados em um lista dentro de um serviço do angular. Como o objetivo é trabalhar com Angular fiz dessa forma, mas é perfeitamente possível conectar este site a um banco de dados real.

O trabalho foi finalizado no dia 11/06/2023

Tema

Escolhi fazer sobre animes simplesmente por ser algo que gosto, peguei trinta animes, seus gêneros e sinopses.

O site mostra diferêntes gêneros e os animes que têm tal gêneros.

Tecnologias

Este site foi feito principalmente angular juntamente com:

  • Poppers
  • Bootstrap
  • jQuery
